# brewlKGI
TSV to pkgi-ps3 .txt files converter
# instructions 
Make 2 folders. One named "tsv", and the other one named "outputtxt". 
Then put .TSV files from a place like NoPayStation, or other websites that host .TSV, into the "tsv" folder.
Run the command in the folder with Brewl.py
After it finishes, go to "outputtxt", and you should have the .txt files!
<python Brewl.py>
